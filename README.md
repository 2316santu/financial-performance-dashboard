Project: Financial Performance Dashboard

Description:
Designed an Excel-based dashboard to analyze financial performance over multiple years. Evaluated key metrics such as revenue, profit, and financial ratios to provide insights into business growth and stability.

Key Features:
- Calculated financial ratios (profit margin, growth rate, liquidity ratios)
- Visualized revenue and profit trends using charts
- Created dashboard for business performance monitoring
- Generated insights for decision-making

Tools Used:
Excel, Data Analysis, Financial Analysis
